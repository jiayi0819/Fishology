==============================================================
	HOW TO RUN FISHOLOGY IN YOUR COMPUTER ?
==============================================================

In [fishology_project] directory:
run "source env/scripts/activate"

In [fishology_django] directory:
run "python manage.py runserver" & leave server runnning

In [fishology_vue] directory:
run "npm run serve" & leave server runnning
